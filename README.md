# Machine Learning Techniques Library
This repository is a collection of practical projects where I explore, implement, and test different ML techniques.

Projects
1. Regression Models (R): Comparison (Decision Tree vs. Random Forest) using rpart and randomForest, evaluated with MSE.

2. SVM Classifier (Python): Implementation of a preprocessing pipeline (RobustScaler, SMOTETomek) to optimize the ROC and AUC of a classifier on unbalanced data.

3. GNN (Python): Implementation of a Graph Neural Network for node classification using PyTorch Geometric, evaluated with accuracy and F1-score. In progress...
